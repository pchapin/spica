The Spica Meta-Library

Spica is a library of libraries of algorithms and data structures written in several languages. The libraries are independent and are separated into subfolders based on the language used. These libraries provide an opportunity to practice implementing both standard, "text book" algorithms and data structures, and to experiment with advanced or unusual algorithms and data structures that are seldom seen in practice, but that are still interesting.

In addition to algorithms and data structures, the Spica libraries sometimes also contain other, potentially useful components in support of timing, unit testing, debugging, and other things. In that respect the libraries are a catch-all for reusable components that I have found useful in my own programming, but that don't seem to fit anywhere else.

This page documents the entire collection of libraries, mostly by pointing to the individual library documentation as appropriate.

The Libraries

Some of the Spica libraries, particularly the ones that are larger or more mature, exist in entirely separate repositories.

  • Ada (separate repository)
  • C (separate repository)
  • C++ (separate repository)
  • Erlang
  • Java
  • Python
  • Ruby
  • Rust

Notes on Building Spica

Since each folder attempts to be an independent subproject, see the README in a folder for more information about building the Spica components in that folder.
